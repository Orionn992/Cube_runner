## About

An early test version of a runner game where you control a cube, dodge obstacles, and race to the finish line.

## Documentation

Build and Run

Build in Unity: File → Build Settings → Build and Run.

Run the build: unzip .zip and run CubeRunner.exe.

Dependencies:

Unity 6000.0.55f1

C# 10

Standard Unity packages
